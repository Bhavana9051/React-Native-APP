# Fubby-APP 🛒📦  

A mobile application where users can buy and sell items, built using React Native and Expo.

---

## Tech Stack Used 🛠️  

Fubby is built with the following technologies:  

- **React Native**: For building the cross-platform mobile UI.  
- **Expo**: To simplify development, testing, and deployment.  
- **Formik & Yup**: For seamless form handling and client-side validation.  
- **React Hooks**: Efficient state management and side effects.  
- **Node.js**: Backend integration and managing dependencies.  
- **Joi**: For strict backend data validation.  
- **Bugsnag**: Real-time error reporting and monitoring.  

---

## Features 🎉  

Fubby provides an interactive marketplace experience:  

- **User Registration & Login**: Secure user authentication and personalized profiles.  
- **Catalog Browsing**: Users can browse items posted by others in a dynamic feed.  
- **Catalog Posting**: Users can post items for sale, including image uploads, item name, price, description, and category selection.  
- **Messaging Interface**: Users can contact sellers directly and negotiate prices.  
- **Account Management**: View your own account info, personal catalogs, and messages.  
- **Notifications**: Real-time updates for messages and item status.

3. **Install dependencies**: `npm install`
4. **Run the app**: `npx expo start`
